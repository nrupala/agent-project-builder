Features

💬

Natural Language Input

Describe what you want to build in plain English. The AI agent understands requirements and generates a complete project.

🌐

Browser-Native GUI

Modern web interface with real-time progress tracking, file tree, code preview, and one-click project download as ZIP.

💻

CLI & Interactive Mode

Full command-line interface for automation, scripting, and CI/CD pipelines. Interactive mode for conversational building.

🔒

Local-First AI

Works with LM Studio or Ollama for 100% offline operation. Your code and prompts never leave your machine.

🛠

Multi-Agent Architecture

Pluggable agent system with configurable capabilities, tools, and behavior. Create custom agents for specific domains.

🔎

Auto Git Integration

Automatic repository initialization, file staging, and meaningful commit messages. Your project is version-controlled from day one.

Get Started in 3 Steps

1

Clone & Install

git clone https://github.com/nrupala/agent-project-builder.git

cd agent-project-builder && npm install

2

Start the Server

npm run server

Open http://localhost:3000 in your browser

3

Describe & Build

Type your project description and click Build. Watch the AI generate your project in real-time.
