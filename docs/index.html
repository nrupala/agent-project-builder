Agent Project Builder

AI-powered software project generation from natural language

What is Agent Project Builder?

Agent Project Builder is an autonomous AI agent system that generates complete software projects from natural language descriptions. Simply describe what you want to build, and our AI agents will create the project structure, write the code, configure dependencies, and set up everything you need to get started.

Key Features

🤖 AI-Powered Generation

Turn natural language descriptions into complete, working software projects using advanced language models.

🔒 100% Local Operation

Run entirely offline using locally stored models from LM Studio or Ollama - no external API calls required.

⚡ Multiple AI Backends

Supports OpenAI (cloud), LM Studio (local), Ollama (local), and VS Code agents for flexible deployment options.

🛠️ Complete Project Generation

Generates full project structures with proper dependencies, configurations, documentation, and build tools.

🧪 Integrated Testing

Automatically runs tests and linting to ensure generated code quality before delivery.

📦 Cross-Platform

Available as standalone executables for Windows, macOS, and Linux, plus Node.js package for developers.

Getting Started

Installation

# Clone the repository
git clone https://github.com/yourusername/agent-project-builder.git
cd agent-project-builder

# Install dependencies
npm install

# Configure your environment (create .env file)
echo "OPENAI_API_KEY=your_key_here" > .env
# Or for local-only mode:
echo "LOCAL_ONLY=true" > .env
echo "MODEL_PROVIDER=lmstudio" >> .env

Usage Examples

# Create a web API from description
node src/index.js -r "Create a REST API for a blog with Node.js and Express"

# Build a mobile app with local models only
node src/index.js -r "Create a fitness tracking app with React Native" -a vscode-agent -m ollama -l

# Start interactive mode
node src/index.js --interactive

Supported Project Types

  • Web Applications: React, Vue, HTML/CSS/JS, Next.js, etc.
  • APIs and Backends: Node.js/Express, Python/Flask, Python/FastAPI, Java/Spring, etc.
  • Mobile Applications: React Native, Ionic, Flutter, etc.
  • Command Line Tools: Node.js CLI, Python CLI, Rust CLI, etc.
  • Libraries and Packages: npm packages, Python packages, etc.
  • Data Processing: ETL pipelines, data analysis scripts, etc.
  • Games: 2D canvas games, text adventures, etc.
  • DevOps: Docker configurations, CI/CD pipelines, infrastructure as code, etc.
